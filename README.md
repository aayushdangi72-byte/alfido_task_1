## Complexity Analysis
- Linked List: Insert/Delete/Search -> O(n)
- Stack: Push/Pop/Top -> O(1)
- Queue: Enqueue/Dequeue/Front -> O(1)
- BST: Insert/Search -> O(log n) average, O(n) worst-case
